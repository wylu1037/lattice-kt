fun ECKeyPair.computeFingerPrint(isGM: Boolean = true): Int

计算密钥对的指纹，该指纹是基于其压缩形式的公钥生成的。

1.获取压缩公钥； 2.对压缩公钥进行SHA-256哈希运算，对哈希执行RIPEMD-160运算，生成160位的哈希值（publicKeyHash）； 3.迭代publicKeyHash的前四个字节，将这四个字节转换成一个32位的整数作为指纹。

Return

指纹

Parameters

isGM

sm2p256v1 or secp256k1
